# ssh-automation

Contents:
- Docker compose file
- ssh.py python script

Usage: 
- First clone the repo.
- create folders for both docker containers.
```
mkdir 1 2 
```
- run docker compose up 
```
docker compose up
```
- install dependencies for ssh.
```
pip3 install paramiko
```
- run ssh.py
```
python3 ssh.py
```

start typing commands and press enter.
